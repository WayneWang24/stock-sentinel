# A股每日分析报告

基于 Megathink 量化分析引擎的每日 A 股全市场扫描报告。

## 在线查看

🔗 [每日分析主页](https://wayneWang24.github.io/a-stock-daily/)
📊 [策略表现追踪](https://wayneWang24.github.io/a-stock-daily/performance.html)

## 内容说明

| 目录 | 内容 |
|------|------|
| `docs/` | GitHub Pages 网站 |
| `docs/api/` | 静态 JSON 数据接口 |
| `reports/` | 每日报告存档 (Markdown + CSV) |
| `strategy/` | 评分方法论与权重变更历史 |

## 每日报告内容

1. **Top 3 最佳买入推荐** — 综合评分最高的 3 只股票
2. **即将启动 Top 100** — 信号置信度最高的 100 只
3. **最低估 Top 100** — 低估评分最高的 100 只
4. **热门板块 Top 10** — 资金净流入最多的板块
