# Megathink 评分方法论

## 概述

Megathink 是一个基于多因子量化分析的 A 股扫描系统，每日自动运行全市场扫描，
输出 4 个核心榜单和综合分析报告。

## 三级过滤架构

### Stage 1: 粗筛 (~5000 → ~3000)
- 排除 ST、停牌、涨停、北交所、次新股
- 最低成交额/成交量过滤

### Stage 2: 中筛 (~3000 → ~500)
- 快速技术指标 (MA/量比/5日涨幅)
- 主力资金方向
- PE/PB 基本面初筛

### Stage 3: 精筛 (~500 → 完整评分)
- 50 维特征工程
- 技术评分 (-100 ~ +100)
- LightGBM 三分类预测
- 信号聚合 + 置信度
- 低估评分 (0-100)
- 筹码分布 + RSRS

## 评分体系

### 技术评分 (-100 ~ +100)
- 趋势 30%: MA 排列 + ADX 方向
- 动量 30%: MACD + KDJ + RSI
- 量能 20%: 量价配合度
- 形态 20%: BOLL 位置 + CCI + 突破

### 低估评分 (0 ~ 100)
- 基本面 50%: PE/PB 分位 + 股息率 + 行业偏离
- 技术面 30%: BOLL 低位 + RSI 低位 + 回撤 + 反转信号
- 资金面 20%: 主力止跌 + 散户恐慌 + 缩量

### ML 预测
- LightGBM 三分类 (上涨/震荡/下跌)
- Walk-forward 验证

### 信号聚合
- 技术评分 40% + ML 概率 60%
- 一致性奖惩机制

## 策略自进化

每周末基于过去 60 天回测结果调整权重:
- 因子 IC (Information Coefficient) 驱动
- 学习率 0.1，单次变化 ≤ 20%
- 权重存储在数据库中，不修改代码

## 回测评估

- T+5: 命中阈值 3%
- T+10: 命中阈值 5%
- T+20: 命中阈值 8%
- 指标: 命中率、平均收益、IC、Top10/30 命中率
